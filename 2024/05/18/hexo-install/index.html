什麼是 Hexo

Hexo 是一款基於 Node.js 的開源部落格框架，使用 Markdown 語言來解析文章，內建很多不同的主題 ，跟內建許多現成的套件可以直接使用。

安裝Hexo

Hexo安裝非常的簡單，但是安裝前要先確定有沒有安裝

  • Node.js
  • Git
  • vscode (自己習慣的編輯器)

Node.js 安裝

前往官網 下載後安裝

git 安裝

前往 git 官網 查看安裝說明

安裝Hexo

安裝完以上套件後執行下面指令

npm install -g hexo-cli

安裝完成後，執行下面語法，Hexo就會自動產生檔案在指定的資料夾裡

安裝完成後執行下面指令就能看到預設的網站

$ hexo server

hexo 相關指令

新增文章

$ hexo new post <文章標題>

新增頁面

$ hexo new page <頁面標題>

產生靜態檔案

$ hexo generate

啟動伺服器

$ hexo server

總結

文章到這邊 hexo 基本已經架設好了，對於其他部落格軟體來說他非常的輕量，他能自動產生靜態網站放在 github 上，對於在寫部落個的又可以省了一筆主機的費用，下一篇再來說明如何透過 github 去架設 hexo
