前言

寫一個互動式的網站，最重要的就是可以將資料存到 mysql 裡面，php 連線的方式有很多種，例如 PDO、mysqli 等等，但是我個人偏好使用 mysqli 所以這篇說明都是使用 mysqli 做連線

如果php 版本使用 8.0 以上， mysqli 沒有預設開啟，所以要前往 php.ini 將 mysqli 開啟
