1. 什麼是 Hexo

Hexo 是一款基於 Node.js 的開源部落格框架，使用 Markdown 語言來解析文章，內建多種主題與套件可使用，能快速產生靜態網站。

2. 安裝 Hexo

安裝 Hexo 前請先準備以下工具：

  • Node.js
  • Git
  • VS Code 或其他編輯器（依個人習慣）
