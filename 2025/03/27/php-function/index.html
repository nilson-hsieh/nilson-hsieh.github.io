不固定長度參數

如果函式輸入參數數量是不固定的，可以使用 php 內建的函式去抓取輸入多少參數及每個參數的值是多少

函數 說明
func_num_args() 取得傳入參數的個數
func_get_args() 取得傳入參數的陣列
func_get_args($i) 取得參數陣列中第 $i 個元素儲存的值（陣列索引從 0 開始）
